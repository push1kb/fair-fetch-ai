# Privacy Policy for Fair Fetch

Effective date: March 4, 2026

Fair Fetch is a Claude Code plugin that surfaces sponsor and ad links in the Claude Code UI.

## Summary

- Fair Fetch runs locally on the user's machine.
- Fair Fetch does not collect personal data.
- Fair Fetch does not use analytics or telemetry.
- Fair Fetch does not sell or share user data.

## What Fair Fetch Processes

Fair Fetch processes only the current URL/page content that Claude Code fetches during user-triggered `WebFetch` or `WebSearch` tool usage, in order to detect known sponsor/ad patterns.

Fair Fetch may make network requests only to:
- the fetched page URL
- supported sponsor/ad provider endpoints needed to resolve attribution details

## Data Collection and Storage

- No account creation
- No personal information collection
- No persistent user tracking
- No remote logging by the plugin itself

Fair Fetch does not maintain a backend service for user data storage.

## Data Sharing

Fair Fetch does not sell, rent, or share personal data with third parties.

## Security

Because Fair Fetch runs locally and does not operate a data backend, data exposure risk from plugin-side storage is minimized.

## Children's Privacy

Fair Fetch is not designed for children and does not knowingly collect personal information from children.

## Changes to This Policy

If this privacy policy changes, updates will be posted in this file with a revised effective date.
